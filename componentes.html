Componentes

🔐 Robot KJ2LR

🎯 Para construir y operar el robot KJ2LR, es esencial contar con una variedad de recursos y componentes que garantizan su funcionalidad y eficiencia, desde herramientas de simulación y diseño como Tinkercad y Cura, hasta dispositivos físicos como impresoras 3D y componentes electrónicos avanzados, cada elemento desempeña un papel crucial. Este documento detalla los recursos necesarios y las características específicas de los componentes que forman parte del KJ2LR, proporcionando una guía clara para su ensamblaje y operación.

💾 Recursos necesarios

NOMBRE

 DESCRIPCIÓN

Simulador TinkerCad
  • Utilizado para la simulación de circuitos y conexiones y modelado 3D.

Multimaker Cura
  • Utilizado para dar dimensiones, relleno y  densidad de los objetos 3D diseñados.

Impresora 3D
  • Utilizada para imprimir los objetos 3D diseñados.

💾 Componentes Totales

IMAGEN NOMBRE DESCRIPCIÓN
Servomotor
  • Facilita el movimiento y/o control de la operación precisa de las pinzas de agarre para manipular objetos de manera eficiente
Arduino Nano
  • Actúa como el "cerebro", coordinando todas las operaciones de los demás componentes que dan interacción al robot autómata KJ2LR.
Módulo Bluetooth HC-06
  • Permite la comunicación inalámbrica entre el robot KJ2LR y dispositivos externos, como smartphones o computadoras, facilitando el control remoto y la transmisión de datos en tiempo real.
Motor paso a paso
  • Facilitan la ejecución de movimientos precisos en las llantas del robot KJ2LR, garantizando un posicionamiento exacto y un control eficiente de su desplazamiento.
Sensor ultrasónico
  • Permiten detectar objetos u obstáculos cercanos al robot KJ2LR, proporcionando datos precisos sobre la distancia y facilitando la navegación autónoma y la toma de decisiones en tiempo real.
Pinzas automatizadas
  • Facilitan el agarre y la manipulación de objetos con precisión, permitiendo al robot KJ2LR realizar tareas de recogida y traslado de forma eficiente y controlada.
Ruedas
  • Permiten el dmovimiento y la dirección del robot KJ2LR, garantizando desplazamientos precisos y estables para una navegación eficiente en diversos entornos.
Chasis
  • Proporciona una estructura sólida y estable para el robot KJ2LR, integrando y soportando todos los componentes como motores, ruedas, sensores y pinzas, asegurando durabilidad y equilibrio durante su operación.
